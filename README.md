# Annuaire de tatoueurs

Projet d’annuaire de tatoueurs réalisé avec **Node.js**, **Express** et **EJS**.

## Objectif

Permettre de :

- consulter une liste de tatoueurs
- voir le détail d’un artiste
- proposer l’inscription d’un tatoueur via un formulaire

- Node.js
- Express
- EJS
- express-validator
- HTML / CSS

## Lancer le projet

```bash
npm install
npm run dev
